## Instructions

Starter template for [NestJS](https://nestjs.com/) with [Prisma](https://www.prisma.io/) includes [GraphQL](https://github.com/nestjs/graphql), [Passport-JWT](https://github.com/nestjs/passport)

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
npm install
```

## Prisma setup

Change the `DATABASE_URL` to your database connection string in `.env` file.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
